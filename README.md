# React Poor Search

This is a poor search component for react.
You upload excel file to make search target.

[Demo Page](https://sashimimochi.github.io/react-poor-search/)

## Usage

example

```js
import ReactPoorSearch from '@sashimimochi/react-poor-search';

function App() {
  return (
    <div className="App">
      <ReactPoorSearch />
    </div>
  );
}

export default App;

```
